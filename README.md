# 포포노트 — 가족이 함께 쓰는 반려동물 다이어리

하루 한 장, 사진을 올리면 AI가 일기를 써주고 가족 모두가 함께 밥·간식·산책을 챙길 수 있는 반려동물 다이어리 앱입니다.

## 주요 기능

- **AI 일기 작성** — 사진 한 장이면 AI가 따뜻한 일기를 작성해줍니다. 수정도 자유롭게, 나만의 일기로 완성하세요.
- **가족 공유** — 초대 코드 하나로 가족 모두가 함께 기록해요. 댓글과 좋아요로 서로의 기록에 반응하세요.
- **케어 기록** — 밥, 간식, 산책을 누가 언제 챙겼는지 한눈에. 가족이 함께 우리 아이의 건강을 관리해요.
- **다이어리 & 앨범** — 캘린더와 피드로 하루하루를 돌아보세요. 월별로 정리되는 사진 앨범.
